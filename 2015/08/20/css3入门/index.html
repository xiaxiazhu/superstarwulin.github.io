随便说的前言

最新的手机和操作系统对最新的css技术支持度越来越好，包括css3、es6 等，网络情况增长也是非常快。css3的动画、flex布局等效果也可以做到主流浏览器上，甚至做到像天猫detail这样高pv数量级的产品上。

这里一些css动画基础

动画，分拆开来讲，就是一些状态和帧的变化。从某种状态过渡到另外一种状态。状态的定义是由某些css，包括不限于：

width：10px 到 100px
padding：0px 到 20px
color：#F00 到 #00F
top：0px 到 10px
border-radius：3px 到 8px
transform：rotate(0deg) 到 ratate(45deg)

理论上来说，任何单独的css属性都可以拿来定义动画。但是那些从无到有，或者不确定的属性值做动画则不生效。

#container p {
display: none;
transition: all 3s ease;
}
#container:hover p {
display: block;
}
/**********************/
#container p {
height: 0px;
transition: all 3s ease;
}
#container:hover p {
height: auto;
}

transform3d，则是dom元素在原有的x，y轴上增加了一个z轴，然后这个通过这个技术实现三维的变化。

translate3d(tx,ty,tz)：他定义了一个3D的位移方法，增加了z轴的偏移量
translateZ(tz)：这个方法只在Z轴偏移，与translateX()和translateY()相似
scale3d(sx,sy,sz)：在原有的scale方法上增加了z轴的参数
scaleZ(sz)：同理，只放大z轴，与scaleX()和scaleY()类似
rotate3d(rx,ry,rz)：将元素以给定参数的某一个轴方向旋转
rotateX(angle)，rotateY(angle)和rotateZ(angle)：只按照某一个轴旋转，    rotate3d(1,0,0,30deg)相当于rotateX(30deg)，
